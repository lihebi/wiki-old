# cmd
* `C-g`: keyboard-quit
* `<Esc><Esc><Esc>`: keyboard-escape-quit
general purpose "get out"

* `C-x C-c`: save-buffers-kill-terminal

* `C-u`: universal-argument
* `C-u 0 C-l`: 当前行top == `C-l C-l`
* `C-u 8 C-f`: 8 times `C-f`
* `C-u 8 *` # 插入8个*
* `C-u 2 C-v`: scroll 2 lines
